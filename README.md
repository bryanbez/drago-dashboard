## Dragos Dashboard

Dragos Dashboard is a web application that allows users to view their Dragos NFTs and their rental status.

## Features

- View your Dragos NFTs
- View your rental status
- View your rental overall income

# Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- Zustand
- SWR

# Using

Google Antigravity and its A.I. Features
